今日は直したバグの数は少ないが、けっこう[ポイントの高いバグ](http://bugzilla.ximian.com/show_bug.cgi?id=75850)を直した。はず。ジェネリクスっていまいちよく分かってないので、実装を直しながら勉強中。なにげにSystem.Xml.dllもほとんど実装しながら勉強していたのだよね…

…と思ってたらバグじゃねーとか言われてますよおい。でもそれだとbeta2と振る舞いが違うぞ…。

昨日ふと思い立って[Poderosa](http://ja.poderosa.org/)をmcsでビルドしようとしたのだけど、まず

```
ValueType vt = (ValueType) 1;
```

みたいなのが[通らず](http://bugzilla.ximian.com/show_bug.cgi?id=75929)挫折。しかし今日みんなRajaがファーストネームだと思っているHarinathが直してくれたので、再挑戦したのだけど、今度は

```
Foo Foo = Foo.GetInstance ();
```

が[通らない](http://bugzilla.ximian.com/show_bug.cgi?id=75941)。型名 == ローカル変数名の時にstatic呼出しという実にピンポイントなコードである。いやはや、まだバグは山積みだ。個々のバグは簡単なものから難しいものまでいろいろあるのだけど。

どうでもいいけど

```
Fix thinko
```

とか書かれると何つーか対応に困りますな…
