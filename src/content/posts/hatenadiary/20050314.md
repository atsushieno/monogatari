## Nullable Type: 'a == b' is true but 'a <= b' is false

Martinがまたハマりそうなコードを見つけてきた。

```
bool ok;  

int? a, b;  

a = null; b = null; ok = a == b;  

Console.WriteLine (ok);  

a = null; b = null; ok = a <= b;  

Console.WriteLine (ok);
```

ECMA的にはa<=bはfalseでなければならないらしい。MSのcscもそんな振る舞いをする。
