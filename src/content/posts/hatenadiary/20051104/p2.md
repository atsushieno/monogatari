そういえばしばらく前にうちのハッカーの面々と話していたのだけど、MethodBase.IsVirtualはなにげに素敵プロパティだ。

```
interface IFoo  

{  

    void Foo ();  

}
```

public class Test : IFoo  
{  
public void Foo () {}

public static void Main ()  
{  
System.Console.WriteLine (typeof (Test)  
.GetMethod ("Foo").IsVirtual);  
}  
}

詳しくはMSDNの説明を参照。
