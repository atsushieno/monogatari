## ■

予告w 明日のエントリーは↓について。

```
CompareInfo ci = CultureInfo.InvariantCulture.CompareInfo;  

string s1 = "\u13C5\u13B5\u13AE\u13B5\u13CD\u13D7";  

string s2 = "\u13A0\u13C2\u13D0\u13A2";  

Console.WriteLine (ci.Compare (s1, s2));
```
